Key Innovation

Neural Dataset Quality Score

The first learnable predictor of dataset-level mAP — before you train a single model.

f(D) = [AQ, IQ, CD, LD, PD, CB]  ∈ ℝ⁶

  AQ — Annotation Quality
       0.6 × completeness + 0.4 × geometry

  IQ — Image Quality
       √(blur_score × noise_cleanliness)

  CD — CLIP Diversity  ← top predictor
       mean pairwise cosine dist (ViT-B/32)

  LD — Lighting Diversity  (entropy)
  PD — Pose Diversity      (entropy)
  CB — Class Balance       (1 − Gini)

DQS(D) = Ridge(PolyFeatures(f(D)))
       → predicted mAP@0.5

Quick Start

Up and running in 5 steps

Requires only Docker Desktop. No GPU needed for the demo.

Install Docker

Download Docker Desktop for Mac, Windows, or Linux.

Clone the repo

git clone https://github.com/ericchen931209/auto-dataset-builder

Start services

docker compose up

First run: ~3 minutes to build.

Open the UI

Navigate to localhost:3000 and create your first dataset.

Evaluate quality

POST /api/v1/datasets/1/evaluate-dqs

Returns DQS score + SHAP feature attribution.
